# Canva制作手順

## 1. テンプレート作成

1. Canvaにログイン
2. 「カスタムサイズ」で各サイズを作成
   - 1080x1080px（正方形）
   - 1080x1920px（縦長）
   - 1200x630px（横長）
3. グリッド表示をON（View → Show rulers and guides）
4. 12カラムグリッド設定

## 2. カラーパレット登録

1. Brand Kit → Colors
2. 以下を登録:
   - `#2563EB` (Primary Blue)
   - `#4F46E5` (Indigo)
   - `#7C3AED` (Purple)
   - `#10B981` (Green)
   - `#FBBF24` (Yellow)
   - `#EF4444` (Red)
   - `#111827` (Gray 900)

## 3. フォント設定

1. Brand Kit → Fonts
2. Noto Sans JP をアップロード（Google Fonts から）
3. ウェイト: Regular, Bold, ExtraBold

## 4. 画像配置

1. Midjourneyで生成した画像をアップロード
2. 各テンプレートに配置
3. サイズ調整・トリミング

## 5. テキスト入力

1. 広告コピー集から適切なコピーを選択
2. フォントサイズ・色を設定
3. 行間調整（1.3-1.5倍推奨）

## 6. CTAボタン作成

1. 長方形ツール
2. 角丸設定（60px）
3. シャドウ追加（Effects → Shadow）
4. テキスト重ねる

## 7. 書き出し

1. Share → Download
2. PNG形式
3. 品質: 高（300dpi推奨、Metaは72dpiでOKだが念のため）
4. 透明背景: OFF

# 制作チェックリスト

## デザイン品質
- [ ] 画像解像度が十分か（1080x1080以上）
- [ ] テキストが読みやすいか（コントラスト比4.5:1以上）
- [ ] フォントサイズが適切か（最小24px以上）
- [ ] マージンが十分か（外周60px以上）
- [ ] ブランドカラーを使用しているか

## Meta広告ポリシー
- [ ] テキストオーバーレイが画面の20%以下か
- [ ] 誇大広告表現がないか
- [ ] 著作権侵害がないか（画像・音楽）
- [ ] CTAが明確か

## 技術仕様
- [ ] ファイルサイズが4GB以下か（通常は数MB）
- [ ] ファイル形式がPNG/JPGか
- [ ] 色空間がRGBか（CMYKはNG）
- [ ] 解像度が推奨値か

## コンテンツ
- [ ] ターゲットに響くコピーか
- [ ] ベネフィットが明確か
- [ ] CTAが行動喚起しているか
- [ ] 安全性訴求が含まれているか
